# assets/

**Static, license-clear design assets** used during composition: fonts, decorative
overlays (particles, light rays, textures), and reference imagery.

- Only assets we have the right to use. Fonts here are the system's display/UI
  families (Cormorant Garamond, Outfit) per licensing.
- Organize by kind, e.g. `fonts/`, `overlays/`, `textures/`, `reference/`.

**Never here:** customer photos (those are per-order, normalized by `AssetService`
at runtime and stored via `StorageService`), licensed logos, or copyrighted imagery.

# Reference image for premium-cinematic-landscape v1

The Gold Standard spec requires an **approved human reference image** paired with each
version (`v1.png`). It is the visual source of truth the QA "brand/masterpiece" gates
calibrate against.

**Milestone note (architecture only):** no image is committed here yet. The approved
look already exists in production as the read-only reference
`design-references/gold-standards/premium-cinematic-landscape-v1.png` (the existing
Premium Cinematic system). When this module moves past architecture-only, a copy of
the approved reference is placed here as `v1.png` and frozen with the version.

Nothing in this module reads or modifies the production file; it is cited only as the
design origin.

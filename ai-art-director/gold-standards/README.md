# gold-standards/

**The visual DNA shared by every product.** Product-agnostic composition, lighting,
palette rules, atmosphere, and print spec. See `../docs/GOLD_STANDARD_SPEC.md`.

- `<id>/schema.json` — JSON Schema every gold standard validates against.
- `<id>/vN.json` — a published, immutable version of the DNA.
- `<id>/vN.png` — the approved human reference image (required; a DNA without an
  approved reference is invalid).

**Never here:** product wording, occasion/sport names, literal fixed palettes,
absolute pixel geometry (use fractions), or runtime code.
